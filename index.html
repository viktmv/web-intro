Invention of the World Wide Web

English scientist Tim Berners-Lee invented the WWW in 1989 while working at CERN.

He also wrote the first web browsing program.

And he invented HTML (1989-1991, first standard - HTML2.0 in 1995)

(CSS came a bit later, 96-98)

Invention of JS

Brendan Eich wrote prototype of the language in 10 days in May 1995. Vastly different from Java. Greatly influenced by Scheme (Lisp) and Self (Smalltalk)

Initially called LiveScript, was released in December 1995

'JavaScript' is trademarked by Oracle

First ECMAScript specification in 1997. ES2, ES3, and finally - ES5 is 2009, ES6 in 2015

Browser Wars

  • Netscape Navigator 95, had over 80% market share. First one to have JavaScript as scripting language. Later, was used as a base for Mozilla Firefox
  • Netscape vs Internet Explorer
  • Firefox vs Internet Explorer
  • Chrome vs All
  • Concept of "rapid releases"

Intro to JS

JavaScript, often abbreviated as JS, is a high-level, dynamic, weakly typed, prototype-based, multi-paradigm, and interpreted programming language.

JavaScript

  • Needs a runtime
  • Single threaded
  • Asynchronous by default
  • Concurrent and non-blocking
  • Is fast!
  • Changing quickly
  • Huge community

Atwood's Law: any application that can be written in JavaScript, will eventually be written in JavaScript
